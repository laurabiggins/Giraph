Adjusting the display


The circles can be dragged around using the mouse to make the display clearer if necessary. A number of options can be selected to change some aspects of the display.

Colours

Changing the colour display

The positions of the circles on the screen are a representation of how closely linked the categories are. The colours are there for aesthetic reasons.

There are 2 options for how the colours are determined in Giraph.

In the default option the colours are determined by the proximity of the circles to each other on the screen.

In the alternative option the colours are not directly determined by the positions of the circles on the screen, but by the proportion of overlapping genes between gene lists. The clustering is calculated independently from the positions of the circles.
These options can be toggled between by selecting the 3 coloured circles button on the tool bar

Changing the number of colours

The number of different colours can be adjusted by selecting the menu option with the small coloured circles as above. This brings up a slider bar which can be used to adjust the r value threshold and increase or decrease the number of colours shown.


Removing categories

A category can be selected by clicking on the circle. This will highlight the circle and the category name will be displayed.

A selected category can be deleted from the display by selecting the 'delete circle' menu item.
This is only deleted from the current view. If the p-value is adjusted, the display is reset.



Adjusting lines

The darker the lines, the greater the proportion of genes that overlap between 2 functional categories.
The number of lines shown can be increased, decreased or all the lines can be removed from the display.



Displaying the category names

Placing the cursor over a circle will highlight it and the category name will be displayed. Clicking on the circle has a similar effect. Clicking on an area of empty space within the display will stop the category name from being displayed.
All of the category names can be displayed by selecting the menu option, and removed by selecting it again.

Changing the p-value threshold

To filter the results by significance, select the p-value icon and enter the required p-value threshold. All the results from the analysis or external file are kept, so the p-value threshold can be increased or decreased.
